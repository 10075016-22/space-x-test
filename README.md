# Monorepo Space X Project

- Backend: Serverless v3 (Python) en `services/backend`
- Infra: AWS CDK (Python) en `infra/cdk`

## Scripts
- `npm run deploy:backend`
- `npm run remove:backend`
- `npm run offline`
- `npm run cdk:bootstrap`
- `npm run cdk:synth`
- `npm run cdk:diff`
- `npm run cdk:deploy`
